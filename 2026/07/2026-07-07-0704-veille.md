### [HIGH] [Microsoft Edge High-Severity Vulnerability Allows Remote Code Execution](https://gbhackers.com/microsoft-edge-high-severity-vulnerability/)
**Source:** vuln gbhackers.com | 2026-07-07
**Thème:** Vulnerabilites

Microsoft a divulgué une vulnérabilité critique de type RCE (CVE-2026-57992) dans son navigateur Edge basé sur Chromium. Cette faille de type Use-After-Free pourrait permettre à des attaquants d'exécuter du code arbitraire sur les systèmes affectés, bien que des conditions spécifiques soient nécessaires pour son exploitation.
